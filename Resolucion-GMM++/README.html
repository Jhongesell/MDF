GMM++ es una libreria para C++ que permite definir diversos tipos de matrices y vectores además de las operaciones para manipular operaciones básicas de álgebra lineal.
Se instala como:
    # apt-get install libgmm++-dev
Para compilar el ejemplo usar:
    $ g++ ejemplito.cpp
Para ejecutar usar:
    $ ./a.out
La facilidad de uso y la gran cantidad de opciones hacen que GMM++ sea una buena opción para trabajar con operaciones elementales de álgebra lineal.
